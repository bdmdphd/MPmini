## Description

These are the files for a 2.5 inch FPV drone I designed. I also designed the frame.

I was looking for a suitable 2.5 inch frame that would fit some current hardware i was planning to recycle, including:

AIO: HDZero Gamme 45A
VTX: DJI O4 air with Flywoo wide lens mod

I wanted the smallest possible 3S build for this.

None of the available 2.5 inch frames out there worked for my needs, so I had no choice but to design my own frame.

The resulting frame is different from all the other frames out there, so the 3D printable accessories will not work on this frame, and I have designed my own 3D printed accessories. With one exception: I made the frame compatible with the Kayoumini library of camera mounts, as I did not want to have to design my own camera mounts, since this is NOT a commercial venture.

I had the frame cut at CNC Madness in T700 carbon fiber.
